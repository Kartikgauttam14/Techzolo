1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd tech-zolo
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install, pnpm install
   ```

3. **Environment Variables:**
   Create a `.env.local` file in the root directory and add necessary environment variables (e.g., database credentials, API keys for Razorpay, EmailJS, etc.).
   ```
   # Example .env.local content
   # NEXTAUTH_URL=http://localhost:3000
   # NEXTAUTH_SECRET=YOUR_NEXTAUTH_SECRET
   # RAZORPAY_KEY_ID=YOUR_RAZORPAY_KEY_ID
   # RAZORPAY_KEY_SECRET=YOUR_RAZORPAY_KEY_SECRET
   # EMAILJS_SERVICE_ID=YOUR_EMAILJS_SERVICE_ID
   # EMAILJS_TEMPLATE_ID=YOUR_EMAILJS_TEMPLATE_ID
   # EMAILJS_PUBLIC_KEY=YOUR_EMAILJS_PUBLIC_KEY
   ```

4. **Database Setup (if applicable):**
   If using a local database, you might need to run migration scripts or seed initial data. Refer to the `scripts/` directory for database-related operations.

## Project Structure Overview

- `app/`: Contains the main application pages, API routes, and global styles.
- `components/`: Reusable UI components, including a `ui/` directory for Shadcn UI components.
- `hooks/`: Custom React hooks for various functionalities (e.g., `use-debounce`, `use-auth`).
- `lib/`: Utility functions and configurations (e.g., authentication, email services, database utilities).
- `public/`: Static assets like images and logos.
- `scripts/`: Backend scripts for database management, API testing, and dependency installation.

## Available API Endpoints

- `/api/auth/`: Authentication related endpoints.
- `/api/contact/`: Contact form submission.
- `/api/godaddy-search/`: Godaddy domain search integration.
- `/api/razorpay/`: Razorpay payment gateway integration.
